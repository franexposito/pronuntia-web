Users

signUp

This method creates an user

Parameters

  • username: String
  • password: String
  • email: String
  • pais: Pointer<Pais> (pass objectId)
  • isFacebook: bool
  • sexo: String
  • pueblo: String
  • provincia: String
  • region: String
  • monigoteBool: bool
  • monigoteElegido: Pointer<Monigote> (pass objectId)
  • imageprofile: Bytes

    • This could be messy, but the subject is: You can choose a monigote (image designed by us, you can see it in the Android version) or you can upload your own image. If you choose a monigote, just pass the objectId that you've fetched before of the monigote to 'monigoteElegido', set 'monigoteBool' true and don't set anything in 'imageprofile'.
    If, instead, we want an image, pass the image as bytes to 'imageprofile', set the 'monigoteBool' false and don't set anything in 'monigoteElegido'.

Success

true

Error

Error Message: error.code + error.message
